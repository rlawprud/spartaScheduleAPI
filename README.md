# 일정 관리 API 프로그램

## 소개
일정을 관리(CLUD)하고, 댓글을 달 수 있는 API 입니다.

## 기능
- 일정 생성
- 일정 수정
- 일정 삭제
- 일정 조회
- 댓글 생성
- 댓글 수정
- 댓글 샂게
- 댓글 조회
